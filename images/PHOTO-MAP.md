# 🗺 Карта використання фотографій

Оригінали — у `images/photos/` (`photo-01.jpg` … `photo-15.jpg`, усі 2048px).
Поруч лежать адаптивні версії `photo-NN-1280.jpg` (1280px) — вони підвантажуються
автоматично через `srcset` на менших екранах заради швидкості; на великих моніторах
використовується повний 2048px-оригінал для максимальної чіткості.

Щоб замінити фото — покладіть новий файл під тим самим іменем
(і, бажано, перегенеруйте 1280-версію, див. нижче).

## Як перегенерувати адаптивні версії (macOS)
```bash
cd images/photos
for f in photo-0*.jpg photo-1*.jpg; do
  [ "${f%-1280.jpg}" = "$f" ] || continue
  sips -Z 1280 -s formatOptions 82 "$f" --out "${f%.jpg}-1280.jpg"
done
```
