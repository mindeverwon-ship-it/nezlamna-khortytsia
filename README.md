# 🌳 Незламна Хортиця

Односторінковий **документальний** scroll-сайт про відродження парку завдяки спільним
зусиллям громади Запоріжжя: від занедбаної території — до живої зеленої зони.

Зроблено як **статичний сайт** (HTML + CSS + JavaScript, без фреймворків і без збірки) —
відкривається подвійним кліком, легко публікується будь-де і просто чіпляється на QR-код.

---

## 🚀 Як відкрити / запустити

**Найпростіше:** двічі клацніть `index.html`.

**З локальним сервером** (рекомендовано):

```bash
cd nezlamna-khortytsia
python3 serve.py            # → http://127.0.0.1:5566
```

---

## 🌐 Як опублікувати (щоб працював QR-код)

QR веде на **адресу сайту в інтернеті**:

- **GitHub Pages** — Settings → Pages → Deploy from branch.
- **Netlify Drop** — перетягніть папку `nezlamna-khortytsia` на app.netlify.com/drop.
- **Vercel** — імпорт репозиторію, framework = "Other".

Після публікації: кнопка **QR** угорі → вставте фінальну адресу → «Оновити QR» → збережіть/роздрукуйте.

## 🖼 Фотографії

Усі **15 реальних фото** лежать у `images/photos/` (`photo-01.jpg` … `photo-15.jpg`, 2048px)
і вже розставлені по розділах документальної історії, hero-мозаїці та галереї — карта в
[`images/PHOTO-MAP.md`](images/PHOTO-MAP.md).

Поруч — адаптивні версії `photo-NN-1280.jpg`: на смартфонах/планшетах через `srcset`
підвантажується легша 1280px-версія (швидкість), на великих моніторах — повний 2048px-оригінал
(максимальна чіткість).

Щоб **замінити** фото — покладіть новий файл під тим самим іменем (і перегенеруйте 1280-версію,
команда — у `images/PHOTO-MAP.md`). Щоб **додати** в галерею — додайте файл і рядок у масив
`gallery` на початку `js/main.js`.

## ✨ Що працює

- **Motion design рівня документального фільму** — Lenis (плавний інерційний скрол) +
  GSAP ScrollTrigger: рух камери (Ken Burns/parallax) на фото, титрова поява тексту (по словах),
  pinned кінематографічна сцена, scrub-анімації. Якщо CDN недоступний — автоматично вмикається
  базовий режим на IntersectionObserver (контент завжди показується);
- **Hero-трейлер з 4 фото** (різні етапи) з повільним рухом, затемненням і глибиною;
- адаптивні фото (`srcset` 1280/2048) — чіткість на великих екранах, швидкість на мобільних;
- липка навігація + прогрес-бар + підсвітка активної секції;
- **документальний таймлайн** із центральним «хребтом», що заповнюється під час скролу;
- підписи під фото у кожному розділі (документальний стиль), без декоративних номерів;
- дорожня карта розділів (швидкий перехід);
- плавна послідовна поява розділів (reveal) + зум фото;
- блок **«Результати спільної праці»** з лічильниками та картками;
- **masonry-галерея** з повноекранним **lightbox** (стрілки, Esc);
- кнопка **QR-коду**;
- повна **адаптивність** (ПК / планшет / смартфон);
- SEO + Open Graph + Schema.org, `lazy-loading`, підтримка `prefers-reduced-motion`.
